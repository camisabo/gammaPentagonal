1. POLIALFABETISMO (Cifrado Multi-Alfabético)
   ✓ Mismo carácter → Diferentes coordenadas según posición
   Ejemplo: 'a' en "aaa" genera [(0,1), (1,12), (2,7), ...]
   Imposibilidad: Análisis de frecuencias tradicional

2. ALTA ENTROPÍA
   ✓ Distribución uniforme en espacio de salida
   X: Media 3.68, Desv.Est 2.70 (rango [0,9])
   Y: Media 8.77, Desv.Est 7.11 (rango [0,19])

3. NO LINEALIDAD
   ✓ Función de grafo: n(j,i) = 2i + j
   ✓ Transformación Y: base + graph + x*2 (mod 20)
   ✓ Tabla Δx: [8,1,1,0,0,1,1,1,3,4,0,6,1,2,4,0,0,2,0,3,1,1]

4. SEGURIDAD PROBABILÍSTICA
   ✓ Trayectorias no predecibles
   ✓ Imposible reconstruir entrada desde fragmento
   ✓ Resistencia a known-plaintext attacks

  ✓ Caminatas discretas sobre vértices
  ✓ Simetrías pentagonales
  ✓ No periodicidad inherente

TOPOLOGÍA:
  ✓ Dodecaedro cúbico (12 caras pentagonales)
  ✓ Coloración de 4 colores (Teorema de los Cuatro Colores)
  ✓ Patrones cuasicristalinos tipo Penrose

CRIPTOGRAFÍA:
  ✓ Cifrado asimétrico probabilístico
  ✓ Seguridad semántica
  ✓ Resistencia a criptoanálisis lineal
  ✓ Confusión y difusión mediante no linealidad
